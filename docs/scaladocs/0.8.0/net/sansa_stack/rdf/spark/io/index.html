  1.  def fromRow(row: Row): Triple

    Converts a Spark SQL Row to a Triple.

    Converts a Spark SQL Row to a Triple.

    row

    the row with columns s,p,o

    returns

    the parsed triple

    Note

    It assumes a row containing exactly 3 columns and the order being s,p,o. Moreover, we assume the following serialization of the RDF entities in the columns which also matches how we read N-Triples into a DataFrame:

    • URI: http://foo.bar
    • bnode: _:123
    • Literal: "123"^^<http://some.datatype.uri>
